# HerramientasdeGestion
SABIA

Sistema de Alfabetización Basado en Inteligencia Artificial

Una plataforma educativa móvil orientada a jóvenes y adultos en situación de rezago educativo. El sistema utiliza inteligencia artificial, actividades interactivas, apoyo visual y auditivo, y una metodología adaptativa para ayudar a los usuarios a desarrollar habilidades de lectura y escritura de manera personalizada.
